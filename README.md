# Expenditure

Small full-stack expenditure tracker project (backend + frontend).

Repository structure
- backend/ — API server
- frontend/ — web client
- frontend-app/ — alternate web client

Prerequisites
- Node.js (LTS) and npm

Quick start
1. Install deps: `npm install` (also run `cd backend && npm install` and `cd frontend && npm install` if needed)
2. Run the project: `./run.sh` (make executable with `chmod +x run.sh`) or start services individually:
   - Backend: `cd backend && npm run dev`
   - Frontend: `cd frontend && npm run dev`

Notes
- Adjust commands to match scripts inside each package if they differ.
- This README is intentionally brief; check package.json files in each folder for exact scripts and environment variables.
